# Análise Empírica de Complexidade

Projeto acadêmico para a disciplina de Estruturas de Dados Básicas I (UFRN).

## 🚀 Como executar

1.  **Compilação e Execução**:
    ```bash
    make run
    ```
    Este comando compila o código C++, executa as medições e gera os gráficos automaticamente.

## 📊 Algoritmos Analisados
- **Busca**: Sequencial e Binária.
- **Ordenação**: Insertion Sort e Quick Sort.

## 🛠️ Requisitos
- Compilador `g++` (padrão C++11 ou superior).
- `Python 3` com bibliotecas `pandas` e `matplotlib` para os gráficos.
